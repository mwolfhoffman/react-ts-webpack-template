# React with Webpack and TypeScript

This is just a react + typescript starter template that uses webpack instead of needing to use create-react-app, remix, next, etc to keep dependencies small.

## Requirements:

1. [node.js](https://nodejs.org/en)
1. [yarn](https://classic.yarnpkg.com/en/)

## To Run:

1. `yarn dev`

## To Build:

1. `yarn build`
